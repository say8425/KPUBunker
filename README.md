# KPU Bunker

KPU Bunker는 한국산업기술대학교의 학생들에 의한 강의평가사이트입니다.
예전에도 있었지만, 인수자를 찾지못해 문닫은 것을 부활시켰습니다.


* Ruby version 
    - 2.2.2

* Ruby on Rails version
    - 4.2.3 

* Database
    - mySQL 5.6.25
    - It can be changed with mongoDB

* Server
    - OS X Server
    - It can be also changed.
